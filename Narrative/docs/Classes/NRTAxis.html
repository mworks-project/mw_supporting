Class Description

NRTAxis.h Narrative

An NRTAxis is one dimensional, and has a coordinate which ranges from 0 to 1, called the "axis coordinate". It simply has a linear dependence on the position along the axis length. For convenience, tick placements can be given in plotting coordinates, which are then transformed to the axis coordinate for drawing by the NRTAxis via a NRTTransform object.
