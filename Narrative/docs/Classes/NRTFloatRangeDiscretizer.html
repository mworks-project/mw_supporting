Class Description

NRTFloatRangeDiscretizer.h Narrative

In this default implementation, a user can choose whether or not to have discrete intervals rounded so that the interval only has a single significant non-zero figure. For example, 1, 10, 500, 400000 are numbers of this type. If this option is used, the discretizer may change the number of points in the range to accommodate for this.
